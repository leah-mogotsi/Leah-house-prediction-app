## House Price Prediction App 

An interactive dweb application that estimates house prices based on property and demographic features. The model analyzes user inputs and provides a predicted property value through a clean Streamlit interface.

## Live Application

Try the app here:
**https://leah-house-prediction-app-bfkv9suqy79kodlrmnfvr6.streamlit.app/**

## Tech Stack

| Tool | Purpose |
|------|---------|
| Python | Core language |
| Pandas & NumPy | Data processing |
| Scikit-learn | Model training & preprocessing |
| Streamlit | Web app deployment |
| Google Colab | Model development |
| GitHub | Version control |
